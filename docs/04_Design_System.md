# ASTRA Design System

> **Design Philosophy:** Premium • Technical • Minimal • Industrial

------------------------------------------------------------------------

# 1. Vision

ASTRA must feel like a professional engineering platform rather than a
consumer chatbot.

Keywords:

-   Premium
-   Precise
-   Calm
-   Trustworthy
-   Modern
-   Industrial

------------------------------------------------------------------------

# 2. Design Principles

-   Dark-first
-   Content first
-   Motion with purpose
-   High information density
-   Clear visual hierarchy
-   Consistent spacing

------------------------------------------------------------------------

# 5. Spacing

Use an 8pt grid.

-   XS = 4dp
-   S = 8dp
-   M = 16dp
-   L = 24dp
-   XL = 32dp
-   XXL = 48dp

------------------------------------------------------------------------

# 6. Corner Radius

-   Chips: 12dp
-   Buttons: 16dp
-   Cards: 24dp
-   Dialogs: 28dp

------------------------------------------------------------------------

# 7. Elevation

Use subtle elevation only.

-   Level 0
-   Level 1
-   Level 3

No excessive shadows.

------------------------------------------------------------------------

Duration:

150ms

250ms

350ms

Animations:

-   Fade
-   Slide
-   Scale
-   Progress

Avoid flashy effects.

------------------------------------------------------------------------

# 16. Settings

Grouped sections.

No long scrolling forms.

------------------------------------------------------------------------

# 17. Glassmorphism

Use lightly:

-   10--15% transparency
-   subtle blur (platform permitting)

Never compromise readability.

------------------------------------------------------------------------

# 18. Accessibility

Minimum touch target: 48dp

Contrast ratio: WCAG AA

Support dynamic font scaling.

------------------------------------------------------------------------

# 21. UI Goal

A user should immediately perceive ASTRA as:

-   an engineering platform,
-   an Edge AI laboratory,
-   a premium professional product,
-   not a generic mobile application.
