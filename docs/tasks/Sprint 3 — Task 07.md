# Sprint 3 — Task 07

# Demo Experience

## Goal

Transform ASTRA into a professional demonstration application by providing a guided demo experience.

The objective is to ensure that any engineer, manager or innovation committee member can understand ASTRA in less than five minutes.

---

## Scope

Implement a dedicated Demo Mode.

---

### 1. Demo Overview

Create a new Demo screen or Dashboard section displaying:

* Device status
* Selected runtime
* Selected model
* Backend status
* Current industry
* Benchmark readiness
* Documents readiness

The screen must immediately communicate whether ASTRA is ready for a live demonstration.

---

### 2. Guided Demo

Provide a guided workflow:

1. Device capabilities
2. Runtime selection
3. Assistant
4. Documents Assistant
5. Benchmark
6. Task Evaluation
7. Model Manager

Display progress through the demonstration.

---

### 3. Demo Readiness

Display readiness indicators:

* Device Ready
* Runtime Ready
* Model Ready
* Benchmark Ready
* Documents Ready

---

### 4. UI

The Demo screen must follow the ASTRA Design System.

---

### Acceptance Criteria

* Demo Mode exists.
* Readiness indicators are displayed.
* Guided flow is functional.
* Android builds.
* iOS builds.

---

feat: add demo experience
